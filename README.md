#### 技术栈
- 前端：vue + element-ui
- 服务端：springboot + mybatis
- 服务端数据库：mysql
- 缓存数据库：redis

#### 技术要点
- 调用nmap扫描端口
- 调用ZAP实现漏洞扫描

### 接口信息
#### 用户接口
- /user/login **用户登录**
- /user/logout **退出登录**
- /user/info **用户信息**
- /user/register  **用户注册**
#### nmap接口
- /nmap/scan  **提交扫描**
- /nmap/result **扫描结果**
